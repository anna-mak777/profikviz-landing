# ПрофиКвиз — грантовый лендинг

Лендинг грантового проекта «ПрофиКвиз» — промышленной викторины для подростков Свердловской области 12–18 лет. Переделка действующего Тильда-лендинга https://proftreker.ru/profiquiz: добавляются живые формы заявок, фирстиль ПРОФТРЕКЕРА, разделы «Новости» и «Фото и видео».

## Статус
Старт проекта — 2026-06-29. Скелет создан, основная работа впереди.

## Как открыть
- `index.html` — боевой лендинг, открывается в браузере двойным кликом.
- `Лендинг ПрофиКвиз - standalone.html` — эталонный wireframe (Claude-артефакт), используется только как визуальный образец, не редактируется.

## Структура
```
.
├── index.html              ← боевой лендинг
├── css/style.css
├── js/main.js
├── images/                 ← встроенные картинки
├── референсы/              ← пакет материалов (брендбук, шрифт, кнопки, выжимка Тильды)
├── памятки/                ← долгосрочные решения и факты
└── CLAUDE.md               ← регламент работы с Claude
```

## Куда уезжает в боевом виде
Лендинг встраивается обратно в Тильду на https://proftreker.ru/profiquiz как HTML-блок. Формы заявок — родные тильдовские (настраиваются в админке Тильды, не в коде).
