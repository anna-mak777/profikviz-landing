# Что лежит в папке `референсы/`

**Эта папка — пакет материалов для разработки лендинга ПрофиКвиз через Claude Code.**
**Когда агент начинает работу — он читает этот файл первым, чтобы понять, что есть и зачем.**

---

## Назначение пакета

Здесь собраны **внешние референсы** и **существующие материалы**, на которые опирается разработка нового лендинга ПрофиКвиз. Папка наполняется на уроке 3.04 курса (заготовка визуального арсенала), используется на уроке 3.06 (создание самого лендинга).

Принцип папки: вместо того чтобы пересказывать агенту словами «у нас есть Тильда-лендинг, цвета вот такие, шрифт хотим вот такой» — кладём всё в файлы, и агент сам читает.

---

## Что внутри (по мере наполнения)

### 📄 `тильда-выжимка.md` *(уже есть)*

Полная выжимка с действующего Тильда-лендинга https://proftreker.ru/profiquiz (собрана 2026-06-22). Содержит:

- главное сообщение проекта и hero-секцию
- описание сути проекта своими словами
- целевую аудиторию (подростки 12–18, Свердловская область)
- структуру текущей страницы по секциям
- все CTA и куда они ведут
- ключевые цифры и факты (80 игр / 2400 участников / 160 карт профессий / 17 отраслей и др.)
- контакты и соцсети
- стиль и тон коммуникации
- секцию **«Что взять отсюда в новый лендинг»** — конкретные указания: что переносим, что усиливаем, что добавляем

**Используется как:** контентная основа нового лендинга. Все факты, цифры, формулировки можно брать отсюда — это утверждённые материалы текущего проекта.

### 📄 `визуальный-референс.md` *(уже есть)*

Ссылка на референс с land-book — лендинг **Consumer App Studio** (https://www.consumerapp.studio/). Анне зашло: стилистика «карточек» (у нас 160 карт профессий — родной формат), композиции «учебных карточек на столе», структура секций образовательного продукта. Внутри файла — подробный анализ стиля оригинала: структура, hero, типографика, микро-приёмы — и **указания, что брать**, а что менять под бренд ПРОФТРЕКЕРА.

**Используется как:** опорный визуальный образец. Применяется по стратегии 1 («сделай как у них»): даём агенту ссылку → агент адаптирует стиль под нашу тему.

### 📄 `кнопки-uiverse.md` *(уже есть, 3 кнопки)*

Три стилистики кнопок под три уровня иерархии CTA:
- **#1 контурная с заливкой** — основной рабочий стиль для большинства CTA («Принять участие», «Зарегистрировать школу»)
- **#2 премиальная с неоновыми блобами** — точечно (1–2 на страницу) для главного hero-CTA и «Стать партнёром-наставником»
- **#3 круглая навигационная со стрелкой** — для слайдера отзывов, «листать к следующей секции», «читать дальше» на карточках

Для каждой кнопки в файле — назначение, готовый HTML (с составной структурой для #2 и #3), CSS, инструкция по адаптации цветов под бренд ПРОФТРЕКЕРА (малиновый/оранжевый вместо исходных синих/зелёных).

**Используется как:** конкретные элементы интерфейса. Применяется по стратегии 2 («вот элемент, используй его»): даём агенту код → агент использует стили на всех кнопках сайта, адаптируя цвета под бренд ПРОФТРЕКЕРА.

### 📄 `шрифт.md` *(уже есть)*

Выбран **Manrope** с Google Fonts. Кириллица проверена через Type tester 2026-06-22 — отображается корректно. В файле — рекомендованные веса для всех уровней (от 400 до 800), готовый код подключения через `<link>`, объяснение почему именно Manrope под ПрофиКвиз и какие веса для какой иерархии (hero H1 → 800, H2 → 700, основной текст → 400 и т.д.).

**Используется как:** прямая инструкция агенту, какой шрифт подключать и как его раздавать по уровням типографики.

### 🖼 `фото/` *(папка готова, наполняется реальными фото ПРОФТРЕКЕРА)*

**Решено НЕ использовать стоковые фото** — Анна положит сюда реальные кадры с мероприятий ПРОФТРЕКЕРА (профориентационные игры, фестивали, посещения предприятий, наставники со школьниками). Для грантового лендинга это сильнее любого стока — реальные подростки Свердловской области, реальные мероприятия, юридическая чистота 100%.

Внутри папки лежит свой `_что-здесь.md` с подробным списком: какие кадры искать в архиве, под какую секцию лендинга каждый, как именовать файлы и **юридический чек-лист** про согласия родителей на использование изображения детей.

**Используется как:** прямые иллюстрации к секциям. Применяется не как референс, а как готовый материал для встраивания в финальный лендинг.

### 🎨 `бренд/` *(уже есть)*

Скопирован официальный фирменный стиль ПРОФТРЕКЕРА из соседнего проекта `proftreker-nastavnik/`:
- `brandbook.txt` — текстовая выжимка для быстрого чтения агентом (главный файл папки)
- `ПРОФТРЕКЕР брендбук презентация.pdf` (2.7 МБ) — визуальный референс с примерами

Внутри папки лежит свой `_что-здесь.md` с инструкцией: что в каком файле и как агенту это применять (палитра, логотип, шрифты). Тот же приём «подгрузка брендбука в папку», что сработал на уроке 3.02 для лендинга `proftreker-nastavnik` — переносим один в один.

## Как работать с этим пакетом (когда начнёшь делать лендинг — урок 3.06)

1. Открой Claude Code в папке `profikviz-лендинг/`
2. В первом сообщении агенту скажи: «У меня в `референсы/` лежит пакет материалов для нового лендинга. Прочитай файл `_что-здесь.md` — там навигация. Изучи всё, потом давай вместе спланируем структуру лендинга»
3. Дай агенту время на чтение всех файлов
4. После этого ставь задачу одним промптом — агент уже будет полностью в контексте
