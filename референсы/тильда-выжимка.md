## 2. О чём проект — суть

Образовательная инициатива в форме интерактивной игры, моделирующей реальный процесс выбора профессии, трудоустройства и адаптации в команде. Помогает молодёжи осознать важность осознанного профессионального выбора в условиях, когда экономика остро нуждается в рабочих кадрах. Участники получают навыки самостоятельного поиска информации о профессиях, образовательных программах и возможностях трудоустройства через работу с уникальным набором карт профессий.

## 4. Структура текущего лендинга — секции по порядку

1. **Шапка** — навигация: Главная, Отзывы, Новости, Проекты, Контакты, Магазин, соцсети ВК
2. **Hero** — название проекта + три метрики (30 / 2400 / 160)
3. **Идея проекта** — описание творческого подхода, методики игры, набора из 160 карт профессий, модели выбора профессии
4. **Задачи проекта** — четыре направления: финальный фестиваль, медиа-освещение, цифровое сопровождение, летние/осенние игры
5. **Презентация проекта** — галерея из 11 изображений (визуальные слайды)
6. **Научно-практический подход** — методология на основе психолого-педагогических разработок и HR-практик
7. **Контакты** — телефон, email, сайт, соцсети
8. **Подвал** — навигация, ссылки на разделы, логотип Tilda
