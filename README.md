# ПрофиКвиз — грантовый лендинг

Лендинг грантового проекта «ПрофиКвиз» — промышленной викторины для подростков Свердловской области 12–18 лет. Переделка действующего Тильда-лендинга `proftreker.ru/profiquiz` на чистом HTML для встраивания обратно в Тильду как HTML-блок.

## Стек
- Чистый HTML + CSS + JS, без сборщиков и фреймворков
- Шрифт Manrope через Google Fonts
- Все CSS-классы с префиксом `pq-` — чтобы не конфликтовать со стилями Тильды
- Картинки и PDF хостятся через GitHub Pages этого репозитория

## Структура
```
.
├── index.html          ← боевой лендинг
├── css/style.css
├── js/main.js
└── images/             ← все картинки и SVG
```

## Локальная разработка
Открыть `index.html` в браузере — никаких зависимостей и сборщиков. Шрифты Manrope подтягиваются с Google Fonts.

## Деплой в Тильду
Лендинг встраивается на страницу Тильды (`proftreker.ru/profiquiz`) как один HTML-блок. Картинки и документы подгружаются с GitHub Pages этого репозитория — править их можно через `git push` без захода в админку Тильды.

Подробная инструкция переноса — в парном приватном репозитории `profikviz-private` (`памятки/перенос-в-тильду.md`).
