# Шрифт для лендинга ПрофиКвиз

## Выбран: Manrope

- **Источник:** [Google Fonts → Manrope](https://fonts.google.com/specimen/Manrope)
- **Лицензия:** Open Font License (свободно для коммерческого использования, включая грантовые проекты НКО)
- **Кириллица:** ✅ проверена через Type tester 2026-06-22 — отображается корректно
- **Тип:** sans-serif, геометричный, современный

## Почему Manrope подходит для ПрофиКвиз

- **Чистая геометрия** — ложится в технологичный, образовательный продукт (промышленная викторина, цифровая платформа, мини-приложение ВК)
- **Подходит для подростковой ЦА** — современный, не «деловой банковский», но и не «детский»
- **Отлично читается на экранах** (для лендинга на разных устройствах — критично)
- **Хорошо стыкуется с типографикой референса** Consumer App Studio (там тоже геометричный sans-serif в духе Inter / Manrope)
- **Отличная кириллица** — все буквы в стилистике латинских, нет «инородных» начертаний

## Рекомендованные веса для лендинга

| Где | Вес | Зачем |
|---|---|---|
| Главный hero-заголовок (H1) | **800** ExtraBold | Максимальный акцент на ключевом сообщении |
| Заголовки секций (H2) | **700** Bold | Чёткая иерархия, чтобы секции легко считывались |
| Подзаголовки (H3) | **600** SemiBold | Внутренние смысловые блоки |
| Основной текст | **400** Regular | Читаемость на длинных абзацах |
| Акценты в тексте (выделения, цифры-метрики) | **500** Medium | Лёгкий смысловой акцент без перегруза |

## Как подключать (для агента на 3.06)

> На лендинге ПрофиКвиз используй шрифт **Manrope** с Google Fonts. Подключи стандартным `<link>` в `<head>` HTML, веса: 400, 500, 600, 700, 800. Кириллица уже проверена — поддерживается. Применяй так:
> - hero H1 → 800
> - H2 секций → 700
> - H3 подзаголовки → 600
> - параграфы → 400
> - выделения, цифры → 500

## Готовый код подключения (агент может скопировать или взять с Google Fonts свежий)

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Manrope:wght@400;500;600;700;800&display=swap" rel="stylesheet">
```

```css
body {
  font-family: 'Manrope', sans-serif;
}
```
