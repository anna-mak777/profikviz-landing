# Фото для лендинга ПрофиКвиз

Эта папка для **реальных фото с мероприятий ПРОФТРЕКЕРА** (профориентационные игры, фестивали, посещения предприятий, групповые занятия со школьниками). Стоковые картинки решено не использовать — у нас есть свой контентный материал, который сильнее любого стока для грантового лендинга.

## Что искать в архиве мероприятий

### Приоритетные кадры

- **Подростки 12–18 в процессе игры** — горизонтальные кадры, в центре кадра один-двое ребят, видны эмоции (азарт, сосредоточенность, радость находки). Подойдёт для **hero-секции** на весь экран.
- **Командная работа за столом с картами профессий** — четверо-шестеро ребят над одним столом, видны карточки. Подойдёт для блока **«Идея проекта»** (160 карт профессий, командно-соревновательный формат).
- **Финальный фестиваль / большая аудитория** — общий план, видна массовость (30 финалистов или больше). Подойдёт для блока **«Задачи проекта»** (финальный фестиваль) или **счётчиков** в hero.
- **Школьник с телефоном / планшетом** — для блока **«Цифровая викторина»** и **«Мини-приложение ВК»** (1600 участников цифровой викторины).
- **Наставник от промышленного предприятия и школьники** — взрослый специалист объясняет что-то группе подростков. Для блока **«Партнёры»** или **«Научно-практический подход»**.
- **Эмоции победителей / награждение** — для блока про 60 «Знатоков профессий».

### Технические требования

- **Горизонтальная ориентация** для большинства секций (hero, фон секций) — это формат, в котором лендинг показывается на десктопе
- **Вертикальные кадры** допустимы для блока **«Цифровая викторина»** (мокап мини-приложения ВК на телефоне)
- **Разрешение от 1920×1080** для hero (полноэкранные фото)
- **Без водяных знаков** фотографа (если есть — попросить версии без)
- **JPG или PNG** — оба формата ок

## Как назвать файлы

Чтобы агент на 3.06 понимал, какое фото для какой секции, лучше класть с осмысленными именами:

- `hero-команда-играет.jpg` — главное фото на hero-секцию
- `карты-профессий-стол.jpg` — для блока «160 карт профессий»
- `финальный-фестиваль.jpg` — для блока «Задачи проекта»
- `цифровая-викторина-телефон.jpg` — для блока «Мини-приложение ВК»
- `наставник-предприятие.jpg` — для блока «Партнёры»
- `награждение-знатоки.jpg` — для блока «60 знатоков профессий»

Если нет времени переименовывать — кидай как есть, я после переименую с понятными именами при сборке лендинга.
