# ПрофиКвиз — грантовый лендинг

## Что это
Лендинг грантового проекта «ПрофиКвиз» — промышленной викторины для подростков Свердловской области 12–18 лет. Переделка действующего Тильда-лендинга https://proftreker.ru/profiquiz на чистом HTML — для встраивания обратно в Тильду как HTML-блок.

## Для кого
- **Образовательные организации** (детские лагеря, школы, колледжи) — ознакомление с проектом и подача заявок на участие
- **Грантодатель** — просмотр документов и отчёта о реализации проекта

## Из каких частей состоит
Одностраничный лендинг. Секции (по порядку): Hero, О проекте, Задачи, Презентация/Галерея, Научно-практический подход, **Новости**, **Фото и видео** (ссылка на ВК-альбом), Документы (включая «Отчёт о реализации»), форма заявки, Контакты.

## Технологии
- Чистый HTML + CSS + JS — никаких фреймворков и сборщиков
- Шрифт Manrope через Google Fonts
- Форма заявки — родная тильдовская (настраивается в админке Тильды)
- За эталон по структуре и компоновке — `Лендинг ПрофиКвиз - standalone.html` (Claude-артефакт, wireframe, не редактируется)

## Структура папок
```
.
├── index.html              ← боевой лендинг
├── css/style.css
├── js/main.js
├── images/                 ← встроенные картинки
├── референсы/              ← брендбук, шрифт, кнопки, фото, выжимка Тильды
└── памятки/                ← долгосрочные решения и факты
```

## Важно про встраивание в Тильду
- Все CSS-классы — с префиксом `pq-` (например, `pq-hero`, `pq-cta`), чтобы не конфликтовать со стилями Тильды
- JS — в IIFE, без глобальных переменных
- Локально `index.html` — полноценная страница для разработки. При встраивании в Тильду содержимое `<body>` переносится в HTML-блок Тильды

## Правила работы с Claude
- Общаемся на русском, простым языком
- Комментарии в коде — на русском
- Перед значимыми изменениями — спрашиваем
- После значимых действий — предлагаем коммит
- Не усложняем — выбираем простые решения
- Если задача большая — сначала Plan Mode, потом работа

## Память
Долгосрочные факты, решения, договорённости — в папке `памятки/`.
Перед сложными задачами — заглянуть в `референсы/_что-здесь.md` за пакетом материалов (брендбук, шрифт, кнопки, выжимка Тильды).

## Безопасность
Секретов в проекте нет — хостинг и формы через Тильду. Если в будущем появятся (например, токен аналитики) — только в `.env`, никогда в коде, и `.env` уже заблокирован в `.gitignore`.
