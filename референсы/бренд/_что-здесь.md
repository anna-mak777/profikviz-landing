# Бренд ПРОФТРЕКЕР

Папка содержит официальный фирменный стиль ООО ПРОФТРЕКЕР. Используется как **источник истины** для цветов, шрифтов, логотипов и общего визуального тона лендинга ПрофиКвиз.

## Что внутри

### 📄 `brandbook.txt`

**Текстовая выжимка из брендбука** — описание фирменного стиля простыми словами. Для агента это **главный файл** в папке: парсить PDF медленно, а текст агент читает мгновенно.

### 📄 `ПРОФТРЕКЕР брендбук презентация.pdf` *(2.7 МБ)*

**Полный официальный брендбук** в виде презентации с визуальными примерами: логотипы (горизонтальный, вертикальный, моноцветный), цветовая палитра с HEX-кодами, шрифтовая система, примеры применения. Используется как **визуальный референс** — когда нужно увидеть «как это выглядит на деле», а не только прочитать описание.

## Как использовать (для агента на 3.06)

> При оформлении лендинга ПрофиКвиз строго соблюдай фирменный стиль ПРОФТРЕКЕРА. Сначала прочитай `референсы/бренд/brandbook.txt` (там описание словами), при необходимости загляни в PDF за визуальными деталями. Применяй:
> - Палитру — тёмный баклажан `#31263B`, малиновый `#FF3770`, оранжевый `#FF9059`, тёплая бежевая (согласно описанию в brandbook.txt)
> - Логотип ПРОФТРЕКЕРА в шапке и/или подвале
> - Фирменные шрифты (если они указаны в брендбуке — иначе использовать Manrope согласно `референсы/шрифт.md`)
> - Не смешивай несколько вариантов логотипа на одной странице
